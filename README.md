# AI Counterfeit Currency Detection System

Deep Learning based counterfeit currency
detection system using:

- MobileNetV2
- FastAPI
- Next.js
- Tailwind CSS
- Modular AI Pipeline

---

# Features

- Real vs Fake Currency Detection
- OCR Serial Number Placeholder
- UV Light Detection Placeholder
- Watermark Detection Placeholder
- Gandhi Face Analysis Placeholder
- Security Thread Detection Placeholder
- Hologram Detection Placeholder
- Denomination Classification Placeholder

uvicorn main:app --reload
