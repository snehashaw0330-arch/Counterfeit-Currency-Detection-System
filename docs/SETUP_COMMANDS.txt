# Counterfeit Currency Detection - Setup Commands

# 0. Prerequisites (one-time, system level)

## OCR engine

The primary OCR engine is **EasyOCR** (deep-learning based,
significantly more accurate than Tesseract on Indian banknote
fonts and the ₹ glyph). EasyOCR is a pip package — installed
by `pip install -r requirements.txt` below.

On first call, EasyOCR downloads its detector + recogniser
weights (~64 MB total) to `%USERPROFILE%\.EasyOCR\model\`.
The FastAPI server pre-warms the reader at startup so the
first /predict request doesn't pay this cost.

## Tesseract OCR (fallback only)

Optional. If Tesseract is installed the backend uses it
silently as a fallback when EasyOCR is unavailable. Skip
this step unless you specifically need the fallback.

### PowerShell (Windows)

winget install --id UB-Mannheim.TesseractOCR --accept-source-agreements --accept-package-agreements --silent

This installs Tesseract to:
C:\Program Files\Tesseract-OCR\tesseract.exe

The backend auto-detects this path. No PATH edit required.

## Node + npm (frontend)

Make sure Node.js 18+ is installed. Verify:

node --version
npm --version

## Python 3.11+

python --version

# 1. Backend Setup

## Create virtual environment (one time)

cd <project-root>
python -m venv venv

## Activate venv

### PowerShell

venv\Scripts\Activate

### Bash / Git Bash

source venv/Scripts/activate

## Install backend dependencies

pip install -r requirements.txt

Notes:
- This installs PyTorch (~500 MB) as an EasyOCR dependency.
- First /predict request after server start triggers a one-
  time ~64 MB model weight download. The startup warmup
  hook does this in a background thread so server boot
  isn't blocked, but the very first request may take 5-10 s
  if it lands before warmup completes.

## Run backend (FastAPI + uvicorn)

uvicorn backend.main:app --host 127.0.0.1 --port 8000

Server: http://127.0.0.1:8000
Swagger Docs: http://127.0.0.1:8000/docs

# Detection Pipeline Overview

The /predict endpoint runs two independent stages and
combines them into a final verdict:

  Stage 1 — MobileNetV2 classifier (224x224, sigmoid)
    output: model_verdict (REAL/FAKE), model_confidence

  Stage 2 — Forensic checks (OpenCV + Tesseract):
    - Structural sanity (aspect, edges, brightness, quadrants)
    - UV light feature detection
    - Watermark gradient
    - OCR serial number
    - Gandhi face (Haar cascade)
    - Security thread (Hough lines)
    - Colour palette integrity (hue entropy + saturation)
    - Denomination OCR
    output: forensic_score (% of checks that passed)

  Combined verdict (model under-weighted because the ML
  classifier over-approves out-of-distribution inputs):
    score = 0.4 * model + 0.6 * forensic
    FAKE  if structural_sanity FAILS  (hard gate)
    REAL  if score >= 0.65 AND pass_count >= 5
                            AND colour palette intact
    FAKE  if score <  0.35 OR (model < 0.35 AND forensic < 0.35)
    else SUSPICIOUS

  Benchmarked on 10 verified real notes + 10 negative-
  control samples: 8/10 real classified correctly,
  7/10 fakes flagged FAKE, 2/10 reverse-side notes
  flagged SUSPICIOUS (model trained mostly on obverse).

# Troubleshooting

## "Cannot find module fastapi" in VSCode

VSCode is pointed at system Python, not the venv. Press
Ctrl+Shift+P → "Python: Select Interpreter" → pick:
.\venv\Scripts\python.exe

## "PSSecurityException" when running npm

PowerShell execution policy is blocking scripts. Either run
npm via Git Bash, or run once in admin PowerShell:
Set-ExecutionPolicy RemoteSigned -Scope CurrentUser

## OCR returns "Tesseract OCR engine not installed"

This message only appears when both EasyOCR AND Tesseract
are unavailable. The primary OCR engine is now EasyOCR
(installed via `pip install -r requirements.txt`). If you
see this message, either:
  - EasyOCR's PyTorch dependency failed to install — re-run
    pip install in a clean venv, or
  - The EasyOCR model download was blocked by the network on
    first use — try a manual reader warmup:
      venv\Scripts\python.exe -c "from backend.forensic import warmup_ocr; print(warmup_ocr())"
Tesseract remains available as a silent fallback if installed.

## Port 8000 already in use

netstat -ano | findstr :8000
taskkill /F /PID <pid> /T
